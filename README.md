# 📦 CRM - Django project


## Description

This is a backend application for managing company sales and analytics.

It allows users to:

- manage products and storage
- create and track sales
- calculate profit
- view analytics (top products, profit over time

---

## 🚀 Features

👤 Users
- authentication and authorization
- roles (owner / employee)
- access limited to user's company

📦 Products
- create and manage products
- assign products to storage

🧾 Sales
- create sales
- add multiple products to a sale
- automatic price calculation with discount

📊 Analytics
- top 5 products by sales
- top 5 products by profit
- profit analytics by date range

---
## 🛠 Tech Stack

- Python
- Django
- Django REST Framework
- PostgreSQL (or SQLite for development)
- Docker / Docker Compose
